# shield-ai

Shield is an AI-powered self-awareness system designed to help users
understand how digital content, notifications, and online interactions
influence their emotions, attention, and decision-making.

Rather than censoring content, Shield focuses on reflection,
context, and psychological insight.

## Core Components

###  Android App
- Behavioral logging (notifications, app usage)
- User reflection & insight cards
- Privacy-first local processing

###  Guardian AI
- NLP-based influence & emotional analysis
- Pattern inference (manipulation, comparison, reassurance, grounding)
- Hybrid architecture (ML classifiers + LLM reasoning)

###  Ethics & Privacy
- No raw content stored long-term
- User-controlled data visibility
- Designed for awareness, not surveillance
